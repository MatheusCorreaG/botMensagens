# Envio de mensagens automáticas

## Descrição
O projeto WhatsApp Bot é uma aplicação desenvolvida em Python que visa simplificar e automatizar a troca de mensagens no WhatsApp. Através de uma interface gráfica intuitiva, o bot permite o envio de mensagens para múltiplos destinatários de forma eficiente e rápida.

### Recursos Principais
- Automação de Mensagens: O bot permite o envio de mensagens para múltiplos contatos de forma automatizada, poupando tempo e esforço.

- Interface Gráfica Amigável: Com uma interface gráfica intuitiva, o bot é acessível para usuários de todos os níveis de habilidade, facilitando a interação e configuração das mensagens a serem enviadas.

- Executável Independente: O projeto pode ser facilmente convertido em um arquivo executável, garantindo portabilidade e facilidade de uso em diferentes plataformas.

## Objetivo
O objetivo principal do WhatsApp Bot é fornecer uma ferramenta prática para profissionais e empresas que precisam lidar com comunicações frequentes via WhatsApp. Ao automatizar o envio de mensagens, o bot ajuda a agilizar processos e melhorar a eficiência das interações com os clientes.
